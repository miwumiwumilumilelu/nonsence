# nonsence

## ⭐⭐⭐ 项目简介

nonsence 是一个 Web 端计算机辅助翻译系统，旨在为用户提供高效、便捷的翻译文本管理服务。

## 🧠 技术选型

### 前端

- 语言及其框架：ts + vue
- 构建工具：vite
- 组件库：arcodesign
- 状态管理库：pinia
- 代码格式化：prettier

### 后端

- 语言以及框架: Python + Django

- 数据库：PostgreSQL
